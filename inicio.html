¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Se utiliza para estructurar y presentar el contenido en la web, incluyendo texto, imágenes, formularios y otros elementos multimedia.

Con HTML, puedes crear fácilmente sitios web interactivos y atractivos.
